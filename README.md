optimisedBigInteger.cpp对第一版代码进行了三个方面的优化：

1.优化后的代码支持负整数的存储和表示，并能对负整数进行运算；

2.第一版代码将大整数以8位十进制位为一段的方式存储和处理处理，优化后以按1位十进制位存储和处理大整数，显著提高了除法运算的效率。

3.修改了一些bug。
